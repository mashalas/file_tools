# drive_space_windows_batch
Windows-batch-скрипт (батник) предназначенный для журналирования свободного пространства на диске.

Подсчёт свободного пространства выполняется в функции drive_free_space, которой как параметр нужно передать букву диска (без двоеточия)

Файл отчёта создаётся в одном каталоге с батником.
