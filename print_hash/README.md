# print_hash
Вычисление контрольных сумм (crc32, md5, sha256 и т.д.) для одного файла, всех файлов каталога или файлов соответствующих маске (с использованием символов подстановки * и ?).

```
print_hash.py <файл|каталог|маска>
```
Если указан каталог - вычисляются контрольные сумммы для всех файлов этого и вложенных каталогов.

По умолчанию вычисляется контрольная сумма crc32. Задать другие алгоритмы вычисления контрольных сумм можно с помощью переменной среды **HASH_ALGO**, в которой указать одно или несколько из следующих значений разделённых запятыми или пробелами:
* crc32
* blake2b
* blake2s
* md4
* md5
* md5-sha1
* mdc2
* ripemd160
* sha1
* sha224
* sha256
* sha384
* sha3_224
* sha3_256
* sha3_384
* sha3_512
* sha512
* sha512_224
* sha512_256
* sm3
* whirlpool

Символы контрольных сумм могут быть разбиты на группы по *N* символов, если установлена переменная **HASH_SPLIT** в которой указывается разрмер группы. 

### Пример:

```
# Вычислить crc32 и sha512 для всех log-файлов в каталоге /tmp , с разбиением символов контрольных сумм на группы по 4 символа:
export HASH_ALGO=crc32, sha512
export HASH_SPLIT=4
print_hash.py /tmp/*.log
```
