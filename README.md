# file_tools
различные утилиты для работы с файлами и каталогами

* ``compare_dirs``
Копирование файлов с раскладыванием файлов по каталогам в соответствии с датой модификации файла.Сравнить содержимое двух каталогов.
___
* ``file2date``
Копирование файлов с раскладыванием файлов по каталогам в соответствии с датой модификации файла.
___
* ``find_by_contents``
shell-скрипт поиска файлов с заданным содержимым
___
* ``fresh_files_processor``
Для новых файлов в заданном каталоге (каталогах) выполнить команды операционной системы.
___
* ``get_files_list``
1.1) Сохранить в файл список файлов до некоторого действия
1.2) Сохранить в файл список файлов после некоторого действия
2) Сравнить различия
3) Подготовить команды воссоздающие изменения
___
* ``hashcopy``
Копирование (перемещение) файлов с добавлением в новом имени файла контрольной суммы этого файла.
___
* ``print_hash``
Вычисление контрольных сумм (crc32, md5, sha256 и т.д.) для одного файла, всех файлов каталога или файлов соответствующих маске (с использованием символов подстановки * и ?).
___
* ``sync_last_dir``
Cинхронизация только одного (последнего) подкаталога в указанном.
___
