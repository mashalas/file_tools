# hashcopy
Копирование (перемещение) файлов с добавлением в новом имени файла контрольной суммы этого файла.

```
hashcopy.py [options] <src> [dst]
```

**src** - исходный файл или каталог
**dst** - целевой каталог; если не указан - оставлять файлы в том же каталоге
**options** - необязательные параметры:

* -a | --algo <имя>  -  имя алгоритма получения контрольной суммы. Допустимы: crc32, md5, 'sha256, 'sha512', sha384, sha1. По умолчанию - crc32
* -n | --name  -  записывать в новом имени файла имя алгоритма получения контрольной суммы
* -m | --move  -  перемещать/переименовывать файлы, а не создавать новые копии
* -r | --recursive  -  рекурсивно с подкаталогами
* -p | --pattern <шаблон>  -  если в качестве источника указан каталог, в этом параметре можно задать маску файлов с использованием символов подстановки "*" (любое количество любых символов) и "?" (один любой символ, но должен присутствовать обязательно)
